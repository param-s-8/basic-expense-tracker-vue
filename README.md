# Basic Expense Tracker Website

Allows you to:
- Add / Remove Income
- Add / Remove Expense
- Check Available Money

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
